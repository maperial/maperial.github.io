new Maperial()

maperial.js, line 32

Instanciate onezz Maperial to build every maps on your web page.

With your Maperial you can now draw Maps and share Data between them.

Example
var maperial = new Maperial();

Examples

Both following examples use this container : <div id="map-container"></div> 

var map = maperial.createMap('map-container');

    container:   'map-container',

    latitude:    53.03787562127988,

});
