A Lens is a view to zoom in your map, centered on where the Lens is placed.

A Lens allows to highlight an area using a greater zoom.

Name Type Description
options object
Name Type Description
width float

The lens width, in pixels. (Default : 250px)
height float

The lens height, in pixels. (Default : 250px)
top float

The lens top from its parent top, in pixels. (Default : placed at parent.bottom - 20px)
left float

The lens left from its parent left, in pixels. (Default : placed at parent.left - 20px)
